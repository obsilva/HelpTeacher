# HelpTeacher
Gerador automático de avaliações.

## Requisitos mínimos do sistema
* SQL Server 2014
* Windows 7 SP1
* Processador de 1 GHz
* 512 MB de memória RAM
* 3 GB de espaço disponível em disco
* Monitor com resolução HD (1280x720)

### Configurações iniciais de desenvolvimento
* Language: C#
* IDE: Visual Studio Community 2017 (15.2.26430.13) with Microsoft .NET Framework (4.7.02046)
* Connector BD: Microsoft SQL Server (native Visual Studio connector)
* Database: SQL Server 2016 Express (13.0.4202)
